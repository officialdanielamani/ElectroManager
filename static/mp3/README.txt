Place the following MP3 sound effect files in this directory:

  scanner.mp3  — played when a QR code or barcode is successfully detected
  success.mp3  — played when a lending or return is submitted successfully
  error.mp3    — played when a scan result is invalid, not found, or submission fails

Sound effects can be enabled or disabled per-user via the Advanced Options panel
on the In/Out page. Files are loaded at runtime; missing files are silently ignored.

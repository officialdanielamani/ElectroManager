Help
System Customization
  • System Name — Shown on the login page (max 64 chars).
  • System Logo — Displayed on the login screen. JPG/PNG, max 1 MB.
  • Banner Timeout — 0 = permanent banner (closes with ✕); 1–60 = auto-dismiss after N seconds.
  • User Signup — Allow self-registration at /register. Disable to require admin-created accounts only.
  • Default Theme — Fallback theme for anonymous users and new accounts.
  • Download All (ZIP) — Control which sections expose a bulk download button.
  • Storage Usage — Two bars showing folder sizes and the fill level of each folder's underlying filesystem. Supports separate volumes (e.g. uploads on HDD, instance on SSD). Bar turns yellow above 70% and red above 80%.
Company Details

Company name, contact info, address, and logo used in printed documents and QR sticker headers.

Currency Settings

Use standard symbols ($, €, £) or currency codes (USD, RM, MYR, SGD). Choose decimal places to match your currency (e.g. 0 for JPY, 2 for USD).

Drawer Size Limits

Sets the maximum rows and columns allowed when creating or editing racks. Prevents excessively large grids (max 32×32).

File Upload Settings
  • Item Uploads — Controls attachments on item pages.
  • Project Uploads — Per-type limits for pictures, documents, schematics, designs, and programs.
  • Share Files — Files accessible via shared links and sticker/icon libraries.
Security Tip: Never allow executable types (.exe, .bat, .sh, .ps1). Settings are saved only when you click Save All Settings.

How It Works
Enable API

Toggling on generates a unique bearer token for your account. Toggling off suspends your key without deleting it.

API Key

Include this token in the Authorization: Bearer <key> header on every API request. Treat it like a password.

Scopes

Scopes are additive — enable only what you need. Each scope is also subject to your role's underlying permissions and the admin's server-wide toggle.

Rate Limit

Current limit: {{ api_rate_limit }} req / sec. Requests over the limit receive HTTP 429 Too Many Requests.

Testing

The Test panel sends a real request to /api/v1/ using your key. Dry Run ON adds dry_run: true to POST requests so nothing is written until you turn it off.
